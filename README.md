# Python Automation Project

## Overview
This project automates email processing using Python.

## Features
- Reads email data from file
- Applies condition-based logic (Send/Skip)
- Generates output file with status

## Technologies Used
- Python
- File Handling

## How to Run
1. Place emails.txt in same folder
2. Run api_test.py
3. Check output.txt
